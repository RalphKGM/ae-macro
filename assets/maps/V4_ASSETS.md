# v0.4 map images

the setup script downloads the public 816x638 map images from the upstream
[quantummacro v0.4 release](https://github.com/QuantumMacro/anime-expeditions/releases/tag/v0.4)
into `assets/maps/v4/`.

the downloaded png files are kept out of this repository because the upstream
project does not currently publish an explicit license. an exact local capture
in `assets/maps/` always overrides an imported image.
